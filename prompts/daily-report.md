# Daily Report Routine

## Trigger
Schedule: Daily 8:00 PM JST

## Task
1. Read all state/*.json files.
2. Generate a concise daily summary:
   - New jobs discovered today.
   - Proposals sent today (with Connects consumed).
   - Active contracts and their status.
   - Revenue received today.
   - Total MTD (month-to-date) revenue.
   - Any issues or items needing human attention.
3. Post summary to Slack.
4. No state file changes. Read-only routine.
